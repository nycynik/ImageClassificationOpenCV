# env setup

Set up using anaconda and openCV

   conda env create -f environment.yml

   -- or -- 

   conda update -n base -c defaults conda
   conda create --name deep python=3

   conda activate deep
   pip install cmake
   pip install numpy
   pip install opencv-contrib-python
   conda install -c conda-forge dlib
